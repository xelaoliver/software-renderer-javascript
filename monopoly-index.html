Monopoly written in HTML and Javascript by Alex Oliver. 16/01/25 - 30/01/25.

This is the demo User Interface ment for testing features and the game. The hand is a temporary image that represents the player and can be moved using the 'Roll Dice' and 'Move 1' buttons. The current street the player is on is displayed in the card to the left of this. You can change the orientation from askew, topdown and rotating. The board is drawn onto the website using a canvas where I can specify where I want and how big I want to draw a polygon. This can create the illusion of 3D. All of the street names, colours, rent, house and hotel prices are fully cusomizable!

I want to twist from the origional game by adding multiple modes: Capitalism, Communism and Anarchy, but I still have to develop the base game (framework) before I start implementing those features. I also want this game to be online so you can play with your friends (if you have any) from different 'school' computers. This will be the most painfull thing to code and also test since I need to have almost 2 tabs/computers on the webpage doing twice the work I usually do.
